Breadth-first search (BFS) is a strategy for running through the vertices of a graph and is a fundamental algorithm of graph theory. It is used to find the shortest path from a given 'source vertex' to any other vertex in the graph (provided there exists such a path). Breadth-first traversal visits all the vertices and edges of graph. A BFS on a graph \(G\) can determine whether \(G\) is connected or not?, compute the connected components of \(G\), compute a spanning forest of \(G\).
In physics literature, Breadth-first search is commonly known as a "burning algorithm" in view of the analogy of a fire burning and spreading through an area, a piece of paper or a fabric, etc. similar to how a breadth first search in a graph proceeds.

What is Breadth First Search ?

Starting from a given source vertex \(v\) of a graph \(G\), we first explore the neighborhood of \(v\) by visiting all vertices that are adjacent to \(v\). We then apply the same strategy to each of the unvisited neighbors of \(v\). The strategy of exploring the neighborhood of a vertex is applied to all vertices of \(G\). The result is a tree rooted at \(v\) and this tree is a subgraph of \(G\). This is a general template for the BFS strategy. The tree resulting from the BFS algorithm is called a Breadth First Search Tree

How does Breadth First Search work ?

Input: A directed or undirected graph \(G=(V,E)\) of order \(n (>0)\). A vertex \(s\) from which the search is to be started. The vertices are numbered from \(1 \dots n\)
Output: A list \(D\) of distances of all vertices from \(s\). A Tree \(T\) rooted at \(s\).
Algorithm:
\(Q \leftarrow [s]\)
\(D \leftarrow [\infty, \infty \dots \infty]\)
\(D[s]\leftarrow 0\)
\(T\leftarrow [ ]\)
\(\textit{while } (length(Q) > 0) \textit{ do }\)
\(v\leftarrow Deque(Q)\)
\(\textit{for each } w \in Adj(v) \textit{ do }\)
\(\textit{if } (D[w]=\infty )\textit{ then }\)
\(D[w] \leftarrow D[v]+1\)
\(Enqueue(Q,w)\)
\(append(T,vw)\)
\(return(D,T)\)
